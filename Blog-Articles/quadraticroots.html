Sum of Two Numbers

By: John Doe

This algorithm uses bases addition operator to add two numbers

  1. Start
  2. Read the coefficients of the equation, a, b and c from the user.
  3. Calculate discriminant = (b * b) – (4 * a * c)
  4. If discriminant > 0: Calculate root1 = ( -b + sqrt(discriminant)) / (2 * a) Calculate root2 = ( -b - sqrt(discriminant)) / (2 * a) Display “Roots are real and different” Display root1 and root2
  5. Else if discriminant = 0: Calculate root1 = -b / (2 *a) root2 = root1 Display “Root are real and equal” Display root1 and root2
  6. Else: Calculate real = -b / (2 * a)
  7. Calculate imaginary = sqrt(-discriminant) / (2 * a)
  8. Display “Roots are imaginary”
  9. Display real, “±” , imaginary, “i”
  10. Stop
