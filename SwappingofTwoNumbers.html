Swapping of Two Numbers

By: Ajay James

This algorithm is used to swap two numbers.

  1. Start
  2. Declare variables a, b and temp
  3. Read values for a and b
  4. Display values a and b
  5. Assign temp = a
  6. Assign a = b
  7. Assign b = temp
  8. Display values a and b after swapping
  9. Stop
